Cataract Detection Algorithm to Classify Cataracts Disease Type Based on Features

[ Research Overview ]

Vision is one of the important senses, with an annual incidence of visual impairment affecting approximately 2 billion individuals. Among them, cataract accounts for 93 million, a large proportion. Also, the problem that has occurred so far is that it takes time for patients to see an ophthalmologist even if they want to be diagnosed quickly. To help doctors make a definitive diagnosis, this study introduces an image processing technique to differentiate between different types of cataracts. In this study, the analysis method used digital image-based image processing technology using MATLAB software. The image processing algorithm consists of several features and aims to classify the posterior cataract and cortical cataract more accurately. The results of the study using the image management method are expected to provide rapid diagnosis to patients by classifying nuclear cataracts, cortical cataracts, and posterior cataracts.



[ Three types of cataracts ]


Each person's iris is a different color, but everyone's pupils are the same color. Because the pupil is circular in shape, it can be easily detected using common circular area detection algorithms, the most common being the Hue transform. Here it is used to automatically detect distinguishable circular pupils. The basic method of the powerful cataract detection algorithm proposed in this paper can be described in three steps: preprocessing, feature extraction, and classification.

Step Description
1. Preprocessing
  • RGB image acquisition
  • Resize the cropped image size to 128*128
  • Identify area of the pupil
  • Remove glare and noise
2. Feature extraction
  • Color quantification and statistical analysis
    • Find the color histogram
    • Calculate the brightness, intensity, uniformity          
    • Calculate the standard deviation of the image          
  • Masking region area
    • Calculate the change rate of divided four area
    • Calculate the accuracy

Global features

Researchers distinguish healthy and abnormal eyes based on parameters like color ratio and uniformity. In identifying abnormality, white hues in the lens area indicate nuclear cataracts, classified using central pupil brightness. Local features refine subdivision for posterior and cortical cataracts.

Local features

The procedure applied to pupil slices, excluding eyebrows or background pixels. Lesion uniformity varies with cataract types, characterized by clustered Gaussian intensity histograms in eye images. Tails in histograms indicate cataract lesions, detectable by comparing with healthy eye histograms. An average filter enhances mode estimation for more accurate lesion detection.
