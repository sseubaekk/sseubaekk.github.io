Cataract Detection Algorithm to Classify Cataracts Disease Type Based on Features


Vision is one of the important senses, with an annual incidence of visual impairment affecting approximately 2 billion individuals. Among them, cataract accounts for 93 million, a large proportion. Also, the problem that has occurred so far is that it takes time for patients to see an ophthalmologist even if they want to be diagnosed quickly. To help doctors make a definitive diagnosis, this study introduces an image processing technique to differentiate between different types of cataracts. In this study, the analysis method used digital image-based image processing technology using MATLAB software. The image processing algorithm consists of several features and aims to classify the posterior cataract and cortical cataract more accurately. The results of the study using the image management method are expected to provide rapid diagnosis to patients by classifying nuclear cataracts, cortical cataracts, and posterior cataracts.
